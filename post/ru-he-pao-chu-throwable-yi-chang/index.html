如何抛出Throwable异常

· 2023-08-08 ·

方法一（强转为子类）

try {
    // ...
} catch (Throwable throwable) {
     if (throwable instanceof BusinessException) {
            throw (BusinessException)throwable;
     }
}

方法二（用运行时异常包裹）

try {
    // ...
} catch (Throwable throwable) {
    throw new RuntimeException(throwable)
}
